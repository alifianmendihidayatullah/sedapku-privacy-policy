Kebijakan Privasi SedapKu

Terakhir diperbarui: 15 Mei 2026

SedapKu adalah aplikasi resep makanan yang membantu pengguna melihat resep, menyimpan resep favorit, membuat perencanaan makanan, dan mengelola daftar belanja.

1. Data yang Kami Kumpulkan

Aplikasi SedapKu tidak mengumpulkan data pribadi pengguna seperti nama, alamat email, nomor telepon, lokasi, kontak, foto, atau informasi identitas pribadi lainnya.

2. Data yang Disimpan di Perangkat

SedapKu dapat menyimpan beberapa data secara lokal di perangkat pengguna, seperti:

  • resep favorit,
  • daftar belanja,
  • data meal planner,
  • preferensi penggunaan aplikasi.

Data tersebut hanya disimpan di perangkat pengguna dan tidak dikirim ke server kami.

3. Pembagian Data ke Pihak Ketiga

SedapKu tidak menjual, membagikan, atau menyewakan data pengguna kepada pihak ketiga.

4. Izin Aplikasi

SedapKu tidak meminta izin sensitif seperti lokasi, kamera, mikrofon, kontak, atau penyimpanan eksternal untuk menjalankan fitur utama aplikasi.

5. Keamanan Data

Karena data disimpan secara lokal di perangkat pengguna, keamanan data juga bergantung pada keamanan perangkat pengguna masing-masing.

6. Anak-anak

SedapKu dapat digunakan sebagai aplikasi umum untuk melihat resep makanan. Aplikasi ini tidak secara sengaja mengumpulkan data pribadi dari anak-anak.

7. Perubahan Kebijakan Privasi

Kebijakan privasi ini dapat diperbarui sewaktu-waktu jika terdapat perubahan fitur atau cara aplikasi menangani data.
